  • Please enter a number of chairs for the table's capacity
  • Then it's time to start adding guests. You can do this by typing a number between 1 and the maximum number of chairs that the table has available
  • If a group has finished eating, you can remove them from the table simply by clicking on any member of that group. Once selected, simply click the 'Remove Group' button. You can remove multiple groups at the same time if you wish.
